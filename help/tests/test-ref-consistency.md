# 标记一致性

此参考可提供有关 Auditor 执行的标记一致性测试的更多信息。

Auditor 的一致性测试旨在查找所有扫描的页面中是否存在不一致的内容。网站上所有页面中的这些值或配置应该是相同的，只有这样，才能确保数据收集是准确的。
